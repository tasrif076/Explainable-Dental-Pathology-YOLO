# Explainable YOLO-based Framework for Dental Pathology Detection

This repository hosts the research implementation of a multi-class dental pathology detection system using **YOLOv8** and **Explainable AI (XAI)** techniques.

## Project Overview
- **Domain:** Medical Imaging / Dental Radiography (OPG X-rays)
- **Model:** YOLOv8l
- **Classes:** Detected 6 different dental conditions.
- **Explainability:** Score-CAM, Feature Map, LIME, Occlusion Heatmap, and SHAP.

## Methodology
The framework focuses on providing both high detection accuracy and transparency in medical diagnosis, making it suitable for clinical decision support.

## Research Status
This work has been submitted to the **IEEE SPICSCON 2026** and is currently under review.
